# Reddit Mod Tools and Migrated Apps Hackathon

**Deadline:** 28 May 2026 @ 4:00am GMT+3
**Prize Pool:** $45,000
**Participants:** 2,717
**Format:** Online / Public / Beginner Friendly

---

## Overview

Reddit is hosting the **Mod Tools and Migrated Apps Hackathon** from April 29 to May 27. $45,000 in prizes for new tools that empower moderators, plus existing mod apps ported from the Data API to **Devvit** (Reddit's Developer Platform).

**The challenge:** Create a utility, automation, or moderation tool that solves existing community pain points using Devvit.

---

## Categories (Grand Prizes)

### 1. New Mod Tool

Build a brand-new utility or tool that makes leading and moderating communities easier. Focus on time savings, thoughtful engagement, and delighting subreddit moderators.

### 2. Ported Data API App

Take a classic Reddit bot (originally built on PRAW or other frameworks) and port it to Devvit. Goal: make essential tools more stable, faster, and easier to install via the App Directory.

> _Eligibility: Existing Data API bots must have operated on Reddit prior to March 2026 and supported one or more communities with 500+ Weekly Active Users. You must be the bot owner or have written permission._

### 3. Moderator's Choice

A panel of moderator judges selects one additional grand prize winner from a shortlist of high-scoring projects in the other two categories.

---

## What to Build

Moderation apps built with Devvit — automated enforcement, queue management, community-building utilities. Apps can have a custom post component or run entirely in the background.

The best apps:

- Reduce moderation load significantly
- Improve community operation
- Incentivize good behavior through innovative utilities
- Are easy to understand, install, and use

**Polish matters:** Apps should be as close to launch-ready as possible. Well-tested, concept-complete projects score higher.

## Judging Criteria

- **Community Impact** — Does it significantly save mod time or improve engagement?
- **Polish** — Close to publishable, Devvit Rules compliant.
- **Reliable UX** — Easy for mods to install, configure, and use at scale.
- **Port Completion** _(port category)_ — How close to replacing the original bot?
- **Ecosystem Impact** _(new tool category)_ — Net new functionality with broad mod appeal?

---

## Submission Requirements

- App listing link on developer.reddit.com
- Reddit usernames of all team participants
- **Tool Overview** — Detailed functionality, capabilities, intended use
- **Project Impact** — 1–3 communities that would benefit and how
- **[Port only] Original Bot username**
- **[Port only] Port Completion notes** — Differences, improvements, gaps vs original
- _[Optional]_ Developer Platform feedback survey
- _[Optional]_ Helper Nomination (community member who helped you)
